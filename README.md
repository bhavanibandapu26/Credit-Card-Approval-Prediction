# 💳 Credit Card Approval Prediction using Machine Learning

A Machine Learning-based web application that predicts whether a credit card application is **Approved** or **Rejected** based on an applicant's financial and personal information. The system is developed using a **Random Forest Classifier** trained on the **Home Credit Dataset** and deployed through a **Flask** web application for real-time prediction.

---

## 🎯 Project Overview

Credit card approval is one of the most important decision-making processes in the banking sector. Manual evaluation of loan applications can be time-consuming, inconsistent, and prone to human error.

This project automates the approval process by analyzing applicant information using Machine Learning techniques and predicting whether an application is likely to be approved or rejected. The application provides a simple, interactive, and user-friendly interface for real-time prediction.

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/bhavanibandapu26/Credit-Card-Approval-Prediction.git
```

### Navigate to Project Directory

```bash
cd Credit-Card-Approval-Prediction
```

### Move to Development Folder

```bash
cd "5. Project Development Phase"
```

### Install Required Packages

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open Browser

```
http://127.0.0.1:5000
```

# 🚀 Usage

1. Start the Flask application.
2. Open the application in your browser.
3. Enter the applicant's information.
4. Click **Predict**.
5. View the predicted approval status and confidence score.
